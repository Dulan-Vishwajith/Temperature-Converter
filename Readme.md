# Temperature Converter

A simple Java console-based program that converts temperatures between Celsius and Fahrenheit. The program accepts a temperature value and a target unit, then performs the conversion and displays the result with formatted output.

## Features
- Converts between Celsius (°C) and Fahrenheit (°F)
- Accepts temperature and target unit as user input
- Displays both input and converted values
- Uses conditional (ternary) operator for clean, concise logic

## Concepts Used
- Java `Scanner` class for user input
- Conditional (ternary) operator for conversion
- Arithmetic operations and formatted output (`printf`)
- String handling and input validation

## Tools & Technologies
- Language: Java SE
- IDE: IntelliJ IDEA / VS Code

## Goal
To strengthen Java fundamentals by building a simple and practical program using user input and conditional logic.
